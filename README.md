# Pokédex

## Pokédex Web React JS

Aplicação React JS e Typescript com testes automatizados. Consumindo dados via PokéApi com axios.

## Tecnologias utilizadas:
  - Yarn
  - React
  - Typescript
  - Redux-Saga

  ## Instalação:

  Comando para instalar as dependências

  ```
  yarn install
  ```

  ```
  yarn install
  ```

  Rodar o projeto utlizando: 
  
  ```
  yarn start
  ```
